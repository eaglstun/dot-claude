# mermaid skill

Add, edit, and validate Mermaid diagrams inside Markdown files.

- `SKILL.md` — entry point: workflow, flowchart essentials, gotchas, validation.
- `references/flowchart.md` — full flowchart syntax.
- `references/diagrams.md` — templates for other diagram types.
- `references/gotchas.md` — escaping, renderer quirks, error decoder.
- `scripts/validate.sh` — parse-check mermaid blocks via `npx mmdc` (no global install).
