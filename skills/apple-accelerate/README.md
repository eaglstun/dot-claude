# apple-accelerate skill

Apple **Accelerate** framework documentation reference — the CPU vectorized-math umbrella
(BLAS/LAPACK, vDSP, vForce, simd, BNNS/BNNSGraph, Sparse Solvers, vImage, Quadrature).

Entry point is [SKILL.md](SKILL.md); the actual docs live in the shared
[`references/apple-accelerate/`](../../references/apple-accelerate/) (repo-wide reference
library). Each reference is condensed from Apple's developer documentation (source URL cited
at the top) and ends with a Gotchas section of the sharp edges.

Standalone framework shelf, like the `rust` and `swift` skills — not tied to a repo. The
**GPU** counterpart is the `apple-silicon` (Metal) skill.
